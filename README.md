# Head & Tail - React App

The project includes three main pages: Home, About, and Head & Tail.

---

## Folder Structure

```
src/
├── App.js
├── index.js
├── pages/
│   ├── Home.js
│   ├── About.js
│   └── HeadTail.js
```

---

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/tejas1505/head-tail-task.git
cd head-tail-task
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```
